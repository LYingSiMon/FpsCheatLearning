# FpsCheatLearning
这是一个 fps 单机游戏逆向练习项目，使用 imgui 绘制

# 游戏获取
CS:起源, 链接：https://pan.quark.cn/s/665474992123

# 功能介绍
- 获取了所有玩家的：坐标、名字、生命值、阵营、id
- 获取了游戏矩阵
- 使用矩阵将玩家坐标转为屏幕坐标
- 使用 imgui 外部绘制，针对敌人和队友，画出不同颜色的方框
